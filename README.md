# 🚀 Ecommerce desde Cero: Guía Completa

¡Bienvenido al repositorio del proyecto! Este es un **E-commerce completo, moderno y de alto rendimiento** desarrollado con el stack de tecnologías más demandado del mercado. 

El proyecto cuenta con un catálogo dinámico, carrito de compras, pasarela de pagos real y un panel de administración robusto para la gestión de productos y pedidos.

---

## 🛠️ Stack Tecnológico

### Frontend & UI
*   **[Next.js](https://nextjs.org/)** - Framework de React con Server-Side Rendering (SSR) y App Router.
*   **[React](https://react.dev/)** - Biblioteca para la construcción de interfaces de usuario.
*   **[Tailwind CSS](https://tailwindcss.com/)** - Framework de CSS orientado a utilidades para un diseño rápido y responsivo.
*   **[Shadcn/ui](https://ui.shadcn.com/)** - Componentes de interfaz accesibles y completamente personalizables.

### Backend & Pagos
*   **[Strapi](https://strapi.io/)** - CMS headless de código abierto para gestionar la API, productos, categorías y pedidos.
*   **[Stripe](https://stripe.com/)** - Pasarela de pago segura para procesar transacciones con tarjeta de crédito/débito.

---

## 🌟 Características Principales

*   ⚡ **Rendimiento Óptimo:** Gracias al renderizado híbrido y optimización de imágenes de Next.js.
*   📱 **Diseño 100% Responsivo:** Adaptado perfectamente a móviles, tablets y ordenadores con Tailwind CSS.
*   🛒 **Carrito de Compras Funcional:** Añadir, eliminar y actualizar cantidades en tiempo real.
*   🔐 **Pasarela de Pagos Segura:** Integración completa con el checkout de Stripe y gestión de webhooks.
*   📦 **CMS Headless:** Panel autoadministrable (Strapi) para control de inventario, stock y multimedia.
*   🎨 **UI Moderna:** Componentes estilizados e interactivos gracias a Shadcn/ui.
